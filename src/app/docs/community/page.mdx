# ⚡ 米米官方伺服器 Rules

## ◆ 核心規則

### ➔ 1. 禁止開小號 *(No Alternate Accounts)*
小號使用嚴格禁止。所有交易、市場操作和轉帳指令都有完整記錄。
※ **違規者終將被發現，主帳號和小號都將被永久封禁。**

### ➔ 2. 禁止自動化濫用 *(No Automation Abuse)*
嚴格禁止使用自動機器人、宏指令、自動農場工具或任何其他自動化遊戲方式。

### ➔ 3. 漏洞回報 *(Exploit Reporting)*
如果您發現漏洞，請立即回報。常見漏洞類型：
- ◇ **漏洞洗錢**：無限刷取油幣或資源
- ◇ **重複抽卡**：重複獲得卡片而不消耗油幣
- ◇ **轉帳漏洞**：異常轉帳操作

**回報方式**：前往 [Support](https://discord.com/channels/812574421465956373/1395821430532210738/1395863009569673237) 開單 詳細描述漏洞並提供證據
▲ **利用漏洞將導致帳號處罰，包括資產清空和永久封禁。**
> ★ 主動回報重大漏洞的玩家將獲得適當獎勵。

### ➔ 4. 尊重他人
禁止人身攻擊、恐嚇或任何基於惡意的行為。請以善意對待他人。

### ➔ 5. 禁止詐騙 *(No Scamming)*
詐騙是嚴格禁止的。這包括從其他玩家那裡拿走卡片或遊戲內貨幣而不履行協議。
> ◆ **注意**：買家後悔不是詐騙。交易完成後即為最終。

### ➔ 6. 禁止遊戲外交易
任何遊戲內資產均不得在遊戲外進行買賣、交易或轉讓。

### ➔ 7. 禁止洗版和垃圾訊息
保持聊天清潔，禁止洗版、大量標記或破壞性行為。

## ◆ 處分機制
```
| 違規程度  | 處分方式 
○ 輕微違規  | 警告或短期禁言 
△ 嚴重違規  | 長期禁言或功能禁用 
● 極嚴重違規 | 永久封禁並清空資產
```
**申訴**：如認為處分有誤，可前往 [Support](https://discord.com/channels/812574421465956373/1395821430532210738/1395863009569673237) 開單 詳細描述申述原因。每次處分最多只有一次申訴機會。

**管理員擁有最終解釋權**：如有爭議或條文模糊之處，由管理員保留最終決定與解釋之權利。
